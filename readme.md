# 🪙 xbar Gold Price Monitor

一个基于 **xbar (BitBar)** 的 macOS 菜单栏黄金价格监控插件。\
支持多家银行实时金价查询，并自动计算当日涨跌。

------------------------------------------------------------------------

## ✨ 功能特性

-   支持 ICBC / CCB / CMB / JD 金价查询
-   菜单栏实时显示当前选中银行价格
-   自动计算相对当日基准价涨跌
-   红涨绿跌显示
-   点击菜单切换银行

------------------------------------------------------------------------

# 🚀 快速上手

## 第一步：安装 xbar

1.  打开官网\
    https://xbarapp.com

2.  下载并安装 xbar

3.  启动后你会在 macOS 菜单栏看到 xbar 图标

------------------------------------------------------------------------

## 第二步：找到插件目录

在菜单栏点击：

xbar → Open Plugin Folder

会打开一个文件夹，这就是插件目录。

------------------------------------------------------------------------

## 第三步：放入脚本文件

将本项目的 Python 脚本文件复制到插件目录，例如命名为：

gold.5m.py

⚠️ 文件名中的 `5m` 表示每 5 分钟刷新一次。

你可以根据需要修改刷新频率：

-   gold.1m.py → 每 1 分钟
-   gold.5m.py → 每 5 分钟
-   gold.10m.py → 每 10 分钟
-   gold.30m.py → 每 30 分钟

------------------------------------------------------------------------

## 第四步：赋予执行权限

打开终端，执行：

chmod +x \~/Library/Application Support/xbar/plugins/gold.5m.py

如果你的插件目录不同，请替换为实际路径。

------------------------------------------------------------------------

## 第五步：刷新插件

在菜单栏点击：

xbar → Refresh All

几秒后，你应该能在菜单栏看到类似：

G 602.31 ▲

点击即可展开查看详细信息。

------------------------------------------------------------------------

# 📌 工作逻辑说明

-   每天首次运行时记录当日基准价
-   所有涨跌均相对该基准价计算
-   每天自动重置
-   数据来自公开接口
